# Ubuntu Wears - Traditional Attire E-commerce

![Ubuntu Wears Logo](https://via.placeholder.com/150)

A full-stack e-commerce platform for traditional African attire built with MERN stack.

## Features

- Product catalog with categories
- User authentication
- Shopping cart
- Admin dashboard
- Payment integration

## Tech Stack

### Frontend
- React.js
- Redux Toolkit
- Tailwind CSS
- Formik & Yup

### Backend
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/imperfectperson-max/ubuntu-wears.git
   cd ubuntu-wears
   ```

2. Set up backend:
   ```bash
   cd server
   npm install
   cp .env.example .env # Update with your credentials
   npm run dev
   ```

3. Set up frontend:
   ```bash
   cd ../client
   npm install
   npm start
   ```
